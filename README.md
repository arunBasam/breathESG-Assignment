# Breathe ESG

ESG ingestion and analyst review platform.

Features:
- Upload records
- Normalize data
- Scope classification
- Approve / Reject workflow
- Audit tracking
- Dashboard analytics
- Multi-tenant architecture
- API versioning

Tech Stack:
- React
- Django REST
- SQLite

Run:

Backend
python manage.py runserver

Frontend
npm run dev

Coverage:
91%
